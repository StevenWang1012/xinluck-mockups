# Xinluck Mockups

給星運 / Xinluck 系統端溝通用的互動 mockup 專案。

## 目前 demo

### 日期欄位加減號上探下鑽

目的：示範營運報表同一張清單如何透過日期欄位標題的 `- / +` 切換時間粒度。

規格重點：

- `+`：往更細一階，年 > 月 > 週 > 日
- `-`：往更粗一階，日 > 週 > 月 > 年
- 欄位標題顯示目前粒度，例如：`日期（週）`
- 清單內日期顯示：
  - 日：`5/1`
  - 週：`5/11`，代表該週週一
  - 月：`5月`
  - 年：`2026年`
- 金額類：依時間區間加總
- 人數類：依時間區間用 player_id / member_id 排重
- 比例類：使用彙總後分子 / 分母重新計算，不直接平均每日比例
